# Interesting Semirings

Simplified Dijkstra semiring: `set`, `plus`, `zero`, `prod`, `one`

## Shortest paths

- `set`: positive integers
- `plus`: addition
- `zero`: 0
- `prod`: minimum operation
- `one`: infinity

## Transitive closure

- `set`: 0,1
- `plus`: and operation
- `zero`: 1
- `prod`: or operation
- `one`: 0

## Maximum Capacity Path

- `set`: positive integers
- `plus`: minimum operation
- `zero`: infinity
- `prod`: maximum operation
- `one`: 0
